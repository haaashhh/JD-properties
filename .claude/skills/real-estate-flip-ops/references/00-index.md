| Interior design boards, mood boards, product library | `design-boards.md` | Designing the design board module, product library, material selections |

**If building the database schema**: read ALL reference files — every entity mentioned becomes a table or field.

**If building a single module**: read only the matching file plus `financial-tracking.md` (finances touch everything).
